# Randomology
A telegram bot to generate random stuff, I built this to chat with my friend by randomly

## Dependency
- pyTelegrambotAPI
- pytest (dev)
- ffmpeg-python
``` shell
python3 -m pip install -r requirements.txt
```

## Testing
run pytest

``` shell
python3 -m pytest
```

## Running
In the project directory
``` shell
python3 bot.py
```
